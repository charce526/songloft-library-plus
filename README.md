# Songloft 歌曲库 Plus

歌曲库 Plus 是一个安装到 Songloft 的 JS 插件，不替换或修改 Songloft 主程序。

## 功能

- 按文件夹、歌手、专辑、流派、年份/年代、语种、风格、来源和格式浏览
- 全局搜索、分类单层浏览、点击表头排序、表格/列表响应式布局
- 点击整行选择、鼠标拖选、移动端长按拖选、Shift 连选，以及全选、反选和取消选择
- 下方居中批量操作浮条支持播放选中内容、收藏、加入歌单、编辑标签和删除
- 仅点击歌曲封面时试听，避免浏览或选择歌曲时误播放
- 单选或批量选择歌曲后一键加入 Songloft 内置收藏
- 普通歌单与电台歌单分区管理，支持创建、编辑、删除、曲目浏览、批量添加和批量移除
- 自动从歌单内带封面的内容随机选取并设置歌单封面
- 智能批量标签编辑：自动显示共同值，以“不同的值”标识差异；只提交实际改动字段
- 文件夹树与歌曲列表左右布局，文件夹栏可整体折叠，深层长名称可横向滚动查看
- 概览集中展示曲库统计、主要歌手、主要专辑、歌单和年份
- 删除曲库记录，可选同时删除本地文件
- 自动/表格/列表视图切换，可设置每页 25–500 首，并记忆显示列、视图、分页数量和文件夹展开状态
- 数字分页、首尾页、上一页/下一页和页码快速定位
- Songloft v2.10.0 起兼容；语种/风格等新能力自动检测

## 构建

```bash
npm install
npm test
npm run build
```

产物会自动包含版本号，例如 `dist/library-plus-v0.2.10.jsplugin.zip`，在 Songloft 的插件管理页面上传并启用即可。

## 开发

```bash
npm run dev
```

开发命令会询问 Songloft 地址与登录信息，并在文件变化后自动构建和上传。

## 安全说明

- 删除本地文件默认关闭，开启后会二次确认。
- 插件内试听不会接管 Songloft 原生播放器队列。
