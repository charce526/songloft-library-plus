# 更新日志

## v0.2.13

- 精简选择浮条按钮文案：播放选中歌曲→播放、加入歌单→歌单、编辑标签→编辑、取消选择→取消，已选择N首→已选N首。
- 修复按钮进入处理状态后图标消失的问题（innerHTML 替代 textContent），移动端不再用文字替换图标，改用 CSS 忙碌状态。
- 调整按钮顺序，将"从歌单移除"移到"删除"之前。
- 播放栏暂停图标从平台不一致的 emoji 替换为跨平台统一的 SVG 图标，通过 CSS class 切换播放/暂停状态。
- 统一选择浮条所有按钮的鼠标悬停提示，补齐缺失项并完善描述。
- 从歌单移除的二次确认从浏览器原生弹窗替换为应用内自定义确认对话框，风格与删除确认一致。
- 修复深色模式下播放栏背景仍为浅色的样式问题（#app 闭合范围扩展到包含播放栏和所有弹窗）。

## v0.2.12

- 修改了深色模式判断逻辑。

## v0.2.11

- 收藏红心改用内联 SVG 图标，修复移动端实心红心变扁、与空心不等宽及 PC 端乱码问题。
- 移动端列表视图缩小复选框与收藏列之间的间距。

## v0.2.10

- 收藏列进一步压缩至 32px，并通过独立定位排序箭头避免额外占用列宽。
- 移除收藏红心悬停时的粉色背景，仅保留红色状态反馈。

## v0.2.9

- 收藏列表头和歌曲红心改为左对齐，排序箭头出现时红心位置保持稳定。
- 收藏列宽度进一步压缩，仅保留容纳红心和排序状态所需的空间。

## v0.2.8

- 将收藏按钮移动到复选框右侧、歌曲封面左侧，表格和紧凑列表保持一致顺序。
- 收藏列表头改为红色心形图标，并支持点击切换收藏优先或未收藏优先排序。
- 收藏排序在完整筛选结果上执行，再进行分页，确保跨页顺序正确。

## v0.2.7

- 收藏红心改用醒目的红色状态。
- 修复在歌单或电台管理页刷新后错误显示歌曲列表的问题。
- 分类详情的“创建歌单”升级为“自动创建歌单”，新建和追加均使用应用内确认弹窗。
- 自动创建歌单时从分类歌曲中随机选择可用封面；已有歌单没有封面时也会自动补充。
- 概览页“我的歌单”移到主要歌手上方，并限制最多显示 8 个歌单。

## v0.2.6

- 播放栏新增可展开的“正在播放”队列，可查看并切换当前播放列表中的歌曲。
- “播放本页歌曲”和“视图”移动到歌曲数量同一行，只在歌曲列表中显示。
- 歌曲表格新增收藏列，支持单曲一键收藏和取消收藏，并同步批量收藏状态。
- 优化时长和大小列宽：时长左对齐，大小列更紧凑。
- 歌手、专辑、流派和年份详情支持一键创建或更新同名歌单，重复歌曲自动跳过。

## v0.2.5

- 选择浮条改为下方居中紧凑显示，播放器开启时自动上移避让。
- 全选、反选、取消选择回到浮条右侧，并通过竖线与批量管理动作分隔。
- 视图设置新增每页 25、50、100、200、500 首选项并记忆设置。
- 分页新增数字页码、首尾页和跳转到指定页。
- “播放当前结果”改为“播放本页歌曲”，仅使用当前页队列；浮条新增“播放选中歌曲”。
- 精简批量标签编辑提示文字。
- 歌单管理新增自动设置封面，随机采用歌单内带封面的歌曲作为封面来源。
- 根据 Songloft 原生 `radio` 类型新增独立电台管理，普通歌单与电台歌单分别管理兼容内容。

## v0.2.4

- 选择浮条改为从出现起即固定浮动，不再挤动歌曲列表或干扰拖选。
- 浮条改用当前主题的主色容器，全选、反选、取消选择移到列表顶栏常驻。
- 文件夹、歌手、专辑、流派、年份等导航改用语义一致的 SVG 图标。
- 修复包含空格和 `&` 等字符的歌手名无法正确筛选的问题，并统一分类值的空白与 Unicode 匹配。
- 移除排序下拉框，表格支持点击标题、歌手、专辑、年份、流派、格式、码率、大小和时长表头排序。
- 深层文件夹长名称不再截断，目录树支持横向滚动并保留完整悬浮提示。
- 扩展音乐库概览，增加分类统计、主要专辑、歌单和年份速览。

## v0.2.3

- 播放器暂停状态改用标准暂停图标，整体界面、文件夹树、歌曲行与歌单卡片更紧凑。
- 统一导航、文件夹和歌曲列表的悬停与选中视觉状态。
- 为选择浮条、提示消息、播放器和所有二级弹窗补充不透明背景兜底。
- 左上角品牌区仅保留“歌曲库 Plus”。
- 加入歌单改为已有歌单与新建歌单二选一，仅选择新建时显示名称输入框。
- 选择操作归集为全选、反选、取消选择，并保留收藏、歌单、标签和删除等批量动作。
- 暂时移除整理本地文件功能与接口。
- 分类浏览改为单层进入，详情页通过左上角返回按钮回到当前分类选择页。

## v0.2.2

- 修复所有设置、标签、整理和歌单弹窗透明及未居中的问题。
- 取消歌曲表头浮动，增强选中行的背景、高亮边线和移动端选中态。
- 分类结果关闭筛选后返回原分类选择页，不再直接显示全部歌曲。
- 新增单曲与批量收藏，自动写入歌曲收藏或电台收藏。
- 搜索框增加清晰边框与聚焦状态。
- 文件夹视图改为左侧目录树、右侧歌曲列表，支持折叠整个目录栏。
- 批量标签编辑自动识别共同值和不同值，仅提交用户实际修改的字段。

- 构建产物继续使用带版本号的文件名。
